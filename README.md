# 26-10-2021

Übungsaufgabe vom 26.10.2020 für die Studierenden des Java I Seminars der Universität zu Köln


### 📝 Aufgabe:
- Importiert dieses Repository in Eclipse. 

Erzeugt eine **neue** Klasse ```Hund```:
- in der ```Hund``` Klasse programmiert ihr eine Methode mit dem Namen ```getName``` die euch als ```return```-Wert einen ```String``` mit einem Hundenamen liefert.

In der Application Klasse:
- Erstellt in der schon vorhandenen ```Application``` Klasse eine ```main```-Methode.
- In der ```main```-Methode erzeugt ihr einen ```Hund```
- Ruft die ```getName```-Methode auf und speichert den Rückgabewert in eine Variable in der ```main```-Methode ab
- Lasst euch die Variable dann in der Konsole ausgeben
